# Typing Carnival

## Overview

Typing Carnival is an interactive web-based typing game built with JavaScript.

The game challenges users to type randomly generated words before the timer runs out. It includes a countdown system, score tracking, sound effects, restart functionality, and real-time input validation.

---

## Technologies and Tools

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Audio API
- GitHub

---

## Features

- Interactive typing gameplay
- Random word generation
- Countdown timer system
- Score tracking
- Sound effects and background music
- Restart game functionality
- Real-time input validation
- Responsive game interface

---

## Project Structure

- index.html
- assets/css/main.css
- assets/js/app.js
- assets/BGM
- assets/image

---

## How to Run

1. Clone the repository
2. Open the project folder
3. Open `index.html` in a browser
4. Click the Start button to begin the game
